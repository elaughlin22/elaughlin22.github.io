What makes me an Empowered Learner?

 

Personal Learning Goals

  • I can track my goals using digital resources, such as Google Calendar for homework.
  • I can think about the learning process by considering what tools work for me and trying new ones.

Build networks

  • I have nearly unlimited tools for research through the Web.
  • There is a network of people I can talk to if I’m stuck, both online and physically.

Using technology to learn and get feedback

  • Tools like Khan Academy are perfect for tracking my learning and for covering concepts in depth.
    • They give feedback on what I can do and what I need to work on.

Understanding fundamental concepts

  • I can use things like Microsoft Office and apply them in various ways.
  • I can use other tools, too — this website was created using JavaScript, CSS, and HTML.
